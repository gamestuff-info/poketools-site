Moves - THIEF

Categories: Damage

An attack that may steal a held item.

Stats

TypeDark
Classspecial class
Power40
Accuracy100
PP10
TargetSelected Pokémon
PriorityNormal

Special Effects

This move has no special effects.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., NO LINK TEXT) and items (e.g., NO LINK TEXT).

  • Blocked by Detect and Protect

    This move will not work if the target has used DETECT or PROTECT this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use MIRROR MOVE to copy it.

Description

Inflicts regular damage. If the target is holding an item and the user is not, the user will permanently take the item. Damage is still inflicted if an item cannot be taken.

This move cannot be selected by METRONOME.

Similar moves

These moves have the same effect, but their stats and effect chances may differ.
