Moves - ASTONISH

Categories: Damage

An attack using a startling shout. It also may make the foe flinch.

Special Effects

    Flags

    • Makes contact

      User touches the target. This triggers some abilities (e.g., STATIC) and items (e.g., NO LINK TEXT).

    • Blocked by Detect and Protect

      This move will not work if the target has used DETECT or PROTECT this turn.

    • Copied by Mirror Move

      A Pokémon targeted by this move can use MIRROR MOVE to copy it.

    Description

    Inflicts regular damage. Has a 30% chance to make the target flinch.

    Power is doubled against Pokémon that have used MINIMIZE since entering the field.
