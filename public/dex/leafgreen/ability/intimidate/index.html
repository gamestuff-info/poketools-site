INTIMIDATE

Lowers the foe's ATTACK.

Description

When this Pokémon enters battle, the opponent's Attack is lowered by one stage. In a double battle, both opponents are affected.

This ability does not take effect when aquired during battle, e.g. with SKILL SWAP

This ability has no effect on an opponent that has a SUBSTITUTE.
