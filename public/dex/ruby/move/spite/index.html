Moves - SPITE

Spitefully cuts the PP of the foe's last move.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used DETECT or PROTECT this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use MIRROR MOVE to copy it.

Description

Lowers the PP of the target's last used move by 4. If the target hasn't used a move since entering the field, if it tried to use a move this turn and failed, or if its last used move has 0 PP remaining, this move will fail.
