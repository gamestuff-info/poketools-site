Moves - Mimic

Categories: Unique

The user copies the target’s last move. The move can be used during battle until the Pokémon is switched out.

Description

This move is replaced by the target's last successfully used move, and its PP changes to 5. If the target hasn't used a move since entering the field, if it tried to use a move this turn and failed, or if the user already knows the targeted move, this move will fail. This effect vanishes when the user leaves the field.

If Chatter, Metronome, Mimic, Sketch, or Struggle is selected, this move will fail.

This move cannot be copied by Mirror Move, nor selected by Assist or Metronome, nor forced by Encore.
