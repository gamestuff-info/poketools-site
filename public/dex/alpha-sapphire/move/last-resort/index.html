Moves - Last Resort

Categories: Damage

This move can be used only after the user has used all the other moves it knows in the battle.

Stats

Classphysical class

Accuracy100
PP5
TargetSelected Pokémon

Special Effects

This move has no special effects.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. This move can only be used if each of the user's other moves has been used at least once since the user entered the field. If this is the user's only move, this move will fail.
