Moves - DISABLE

Categories: Unique

Psychically disables one of the foe's moves.

Special Effects

  • Inflicts Disable on the target(s).
  • Lasts 4 turns.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used DETECT or PROTECT this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use MIRROR MOVE to copy it.

Description

Disables the target's last used move, preventing its use for 2-5 turns, selected at random, or until the target leaves the field. If the target hasn't used a move since entering the field, if it tried to use a move this turn and failed, if its last used move has 0 PP remaining, or if it already has a move disabled, this move will fail.
