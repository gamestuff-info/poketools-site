Moves - SKILL SWAP

Categories: Unique

The user swaps special abilities with the target.

Stats

TypePsychic
Classspecial class
Power–
Accuracy100
PP10
TargetSelected Pokémon
PriorityNormal

Special Effects

This move has no special effects.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used DETECT or PROTECT this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use MIRROR MOVE to copy it.

Description

User and target switch abilities. Ignores accuracy and evasion modifiers.

If either Pokémon has WONDER GUARD, this move will fail.
