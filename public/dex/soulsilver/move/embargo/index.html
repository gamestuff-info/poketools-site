Moves - Embargo

Categories: Ailment

It prevents the foe from using its held item. Its Trainer is also prevented from using items on it.

Stats

TypeDark
Classstatus class
Power–
Accuracy100
PP15
TargetSelected Pokémon
PriorityNormal

Special Effects

  • Inflicts Embargo on the target(s).
  • Lasts 5 turns.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Reflectable

    This move may be reflected back at the user with Magic Coat or NO LINK TEXT.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Target cannot use its held item for five turns. If the target leaves the field, this effect ends.

If a Pokémon under this effect uses Bug Bite or Pluck on a Pokémon holding a berry, the berry is destroyed but not used. If a Pokémon under this effect uses Fling, it will fail.

This effect is passed by Baton Pass.
