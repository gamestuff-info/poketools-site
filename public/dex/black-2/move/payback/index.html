Moves - Payback

Categories: Damage

If the user moves after the target, this attack’s power will be doubled.

Stats

TypeDark
Classphysical class
Power50
Accuracy100
PP10
TargetSelected Pokémon
PriorityNormal

Special Effects

This move has no special effects.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. If the target uses a move or switches out this turn before this move is used, this move has double power.
