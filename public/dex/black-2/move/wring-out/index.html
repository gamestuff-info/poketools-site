Moves - Wring Out

Categories: Damage

The user powerfully wrings the target. The more HP the target has, the greater this attack’s power.

Stats

TypeNormal
Classspecial class
Power–
Accuracy100
PP5
TargetSelected Pokémon

Special Effects

This move has no special effects.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. Power directly relates to the target's relative remaining HP, given by 1 + 120 * current HP / max HP, to a maximum of 121.
