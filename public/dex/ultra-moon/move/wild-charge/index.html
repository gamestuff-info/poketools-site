Moves - Wild Charge

Categories: Damage

The user shrouds itself in electricity and smashes into its target. This also damages the user a little.

Stats

TypeElectric
Classphysical class
Power90
Accuracy100
PP15
TargetSelected Pokémon
PriorityNormal

Special Effects

  • User takes 25% of damage given as recoil.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. User takes 1/4 the damage it inflicts in recoil.
