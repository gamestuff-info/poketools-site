Moves - Echoed Voice

Categories: Damage

The user attacks the target with an echoing voice. If this move is used every turn, it does greater damage.

Stats

TypeNormal
Classspecial class
Power40
Accuracy100
PP15
TargetSelected Pokémon
PriorityNormal

Special Effects

This move has no special effects.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

  • Sound-based

    Pokémon with Soundproof are immune to this move.

  • Ignores Substitute

    This move ignores the target's Substitute.

Description

Inflicts regular damage. If any friendly Pokémon used this move earlier this turn or on the previous turn, that use's power is added to this move's power, to a maximum of 200.
