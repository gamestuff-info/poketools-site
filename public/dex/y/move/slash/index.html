Moves - Slash

Categories: Damage

The target is attacked with a slash of claws or blades. Critical hits land more easily.

Stats

TypeNormal
Classphysical class
Power70
Accuracy100
PP20
TargetSelected Pokémon
PriorityNormal

Special Effects

  • Critical hit rate increased by 1%.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. User's critical hit rate is one level higher when using this move.
