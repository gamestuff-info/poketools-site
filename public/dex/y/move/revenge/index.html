Moves - Revenge

Categories: Damage

An attack move that inflicts double the damage if the user has been hurt by the opponent in the same turn.

Stats

TypeFighting
Classphysical class
Power60
Accuracy100
PP10
TargetSelected Pokémon
Priority-4

Special Effects

This move has no special effects.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. If the target damaged the user this turn and was the last to do so, this move has double power.

Pain Split does not count as damaging the user.

These moves have the same effect, but their stats and effect chances may differ.
