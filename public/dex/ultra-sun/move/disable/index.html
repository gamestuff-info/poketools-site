Moves - Disable

Categories: Unique

For four turns, this move prevents the target from using the move it last used.

Stats

TypeNormal
Classstatus class
Power–
Accuracy100
PP20
TargetSelected Pokémon
PriorityNormal

Special Effects

  • Inflicts Disable on the target(s).
  • Lasts 4 turns.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Reflectable

    This move may be reflected back at the user with Magic Coat or Magic Bounce.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

  • Ignores Substitute

    This move ignores the target's Substitute.

Description

Disables the target's last used move, preventing its use for 4 turns, selected at random, or until the target leaves the field. If the target hasn't used a move since entering the field, if it tried to use a move this turn and failed, if its last used move has 0 PP remaining, or if it already has a move disabled, this move will fail.
