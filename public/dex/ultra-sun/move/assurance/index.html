Moves - Assurance

Categories: Damage

If the target has already taken some damage in the same turn, this attack’s power is doubled.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. If the target takes damage this turn for any reason before this move is used, this move has double power.
