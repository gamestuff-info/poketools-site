Moves - WRAP

Categories: Ailment, Damage

Squeezes the foe for 2-5 turns.

Stats

TypeNormal
Classphysical class
Power15
Accuracy85
PP20
TargetSelected Pokémon
PriorityNormal

Special Effects

  • Inflicts Trap on the target(s). (100% chance)
  • Lasts 5-6 turns.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., NO LINK TEXT) and items (e.g., NO LINK TEXT).

  • Blocked by Detect and Protect

    This move will not work if the target has used DETECT or PROTECT this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use MIRROR MOVE to copy it.

Description

Inflicts regular damage. For the next 2–5 turns, the target cannot leave the field and is damaged for 1/16 its max HP at the end of each turn. The user continues to use other moves during this time. If the user leaves the field, this effect ends.

Has a 3/8 chance each to hit 2 or 3 times, and a 1/8 chance each to hit 4 or 5 times. Averages to 3 hits per use.

RAPID SPIN cancels this effect.
