Moves - Fling

Categories: Damage

The user flings its held item at the target to attack. Its power and effects depend on the item.

Stats

TypeDark
Classphysical class
Power–
Accuracy100
PP10
TargetSelected Pokémon
PriorityNormal

Special Effects

This move has no special effects.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. Power and type are determined by the user's held item. The item is consumed. If the user is not holding an item, or its item has no set type and power, this move will fail.

This move ignores Sticky Hold.

If the user is under the effect of Embargo, this move will fail.
