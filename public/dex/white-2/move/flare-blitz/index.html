Moves - Flare Blitz

Categories: Ailment, Damage

The user cloaks itself in fire and charges at the target. The user sustains serious damage and may leave the target burned.

Stats

TypeFire
Classphysical class
Power120
Accuracy100
PP15
TargetSelected Pokémon
PriorityNormal

Special Effects

  • Inflicts Burn on the target(s). (10% chance)
  • User takes 33% of damage given as recoil.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Copied by Mirror Move

    A Pokémon targeted by this move can use Mirror Move to copy it.

  • Defrosts when used

    This move can be used while frozen to force the Pokémon to defrost.

Description

Inflicts regular damage. User takes 1/3 the damage it inflicts in recoil. Has a 10% chance to burn the target. Frozen Pokémon may use this move, in which case they will thaw.
