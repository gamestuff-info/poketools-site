Moves - Me First

Categories: Damage

The user tries to cut ahead of the target to steal and use the target’s intended move with greater power.

Stats

Special Effects

This move has no special effects.

Flags

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Ignores Substitute

    This move ignores the target's Substitute.

Description

If the target has selected a damaging move this turn, the user will copy that move and use it against the target, with a 50% increase in power.

If the target moves before the user, this move will fail.

This move cannot be copied by Mirror Move, nor selected by Assist, Metronome, or Sleep Talk.
