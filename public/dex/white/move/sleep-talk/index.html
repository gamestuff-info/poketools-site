Moves - Sleep Talk

Categories: Unique

While it is asleep, the user randomly uses one of the moves it knows.

Special Effects

This move has no special effects.

Flags

No special flags apply to this move.

Description

Only usable if the user is sleeping. Randomly selects and uses one of the user's other three moves. Use of the selected move requires and costs 0 PP.

If the selected move requires a recharge turn and the user is still sleeping next turn, then it's forced to use this move again and pay another PP for the recharge turn.

This move cannot be copied by Mirror Move, nor selected by Assist, Metronome, or Sleep Talk.
