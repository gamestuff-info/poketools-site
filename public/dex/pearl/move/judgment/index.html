Description

Inflicts regular damage. If the user is holding a plate or a drive, this move's type is the type corresponding to that item.

Note: This effect is technically shared by both NO LINK TEXT and Judgment; however, Techno Blast is only affected by drives, and Judgment is only affected by plates.
