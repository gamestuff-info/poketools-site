Moves - Karate Chop

Categories: Damage

The target is attacked with a sharp chop. Critical hits land more easily.

Flags

  • Makes contact

    This move will not work if the target has used Detect or Protect this turn.

    A Pokémon targeted by this move can use Mirror Move to copy it.

Description

Inflicts regular damage. User's critical hit rate is one level higher when using this move.
