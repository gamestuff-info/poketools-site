Moves - Captivate

Categories: Net Good Stats

If any opposing Pokémon is the opposite gender of the user, it is charmed, which harshly lowers its Sp. Atk stat.

Description

Lowers the target's Special Attack by two stages. If the user and target are the same gender, or either is genderless, this move will fail.
