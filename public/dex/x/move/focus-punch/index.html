Moves - Focus Punch

Categories: Damage

The user focuses its mind before launching a punch. This move fails if the user is hit before it is used.

Stats

TypeFighting
Classphysical class
Power150
Accuracy100
PP20
TargetSelected Pokémon
Priority-3

Special Effects

This move has no special effects.

Flags

  • Makes contact

    User touches the target. This triggers some abilities (e.g., Static) and items (e.g., Sticky Barb).

  • Blocked by Detect and Protect

    This move will not work if the target has used Detect or Protect this turn.

  • Punch-based

    This move has 1.2× its usual power when used by a Pokémon with Iron Fist.

Description

Inflicts regular damage. If the user takes damage this turn before hitting, this move will fail.

This move cannot be copied by Mirror Move, nor selected by Assist, Metronome, or Sleep Talk.
