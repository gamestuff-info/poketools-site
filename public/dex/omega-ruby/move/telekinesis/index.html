Moves - Telekinesis

Categories: Ailment

The user makes the target float with its psychic power. The target is easier to hit for three turns.

Description

For three turns (including this one), moves used against the target have 100% accuracy, but the target is immune to Ground damage. Accuracy of one-hit KO moves is exempt from this effect.

This effect is removed by Gravity. If Gravity is already in effect, this move will fail.
